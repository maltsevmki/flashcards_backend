# Security Policy

## Supported Versions

This project is currently in **early development**. No official release has been published yet.
Security updates and patches will be provided for the latest codebase only, until a stable release is published.

| Version         | Supported |
|-----------------|:---------:|
| unreleased      | ✔️        |
| < first release | ❌        |

> Once the first stable version is released, this table will be updated with version support details.

## Reporting a Vulnerability

If you discover a security vulnerability, please **report it privately** using [GitHub's security advisory feature](https://github.com/maltsevmki/flashcards_backend/security/advisories/new).

- **Do not** open public issues for security concerns or vulnerabilities.
- Responsible disclosure helps protect all users while issues are being resolved.

Thank you for helping to keep this project secure!
